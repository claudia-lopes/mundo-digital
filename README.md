# mundo-digital



📟 DigiVirtual v2.0 | Explorador de Dados
Uma interface moderna e responsiva de acesso ao Mundo Digital, desenvolvida para consulta e visualização de dados da franquia Digimon. Este projeto demonstra habilidades em Front-end moderno, consumo de APIs REST e design focado na experiência do usuário (UX).

🚀 Tecnologias Utilizadas
HTML5 & CSS3: Estrutura e estilização customizada.

Tailwind CSS: Utilitários de design para agilidade e responsividade.

Bootstrap 5: Estrutura de grid e componentes de UI (Modais).

JavaScript (ES6+): Manipulação dinâmica do DOM, consumo de API assíncrona (async/await) e filtros em tempo real.

DigiAPI: Interface pública de dados utilizada para alimentar o sistema.

🛠️ Funcionalidades Técnicas
Renderização Dinâmica: Os cards são gerados automaticamente a partir de objetos JSON.

Filtro em Tempo Real: Sistema de busca inteligente que filtra os resultados conforme a digitação, sem recarregar a página.

Tratamento de Exceções: Implementação de Placeholders e Image Proxy (wsrv.nl) para garantir que imagens com bloqueio de CORS ou links quebrados não comprometam a interface.

Interface Futurista: Design inspirado em interfaces de comando e monitoramento, com efeitos de Glassmorphism e animações de Scanline.

🎨 Design & UX
A interface foi projetada com uma estética "Dark Tech", utilizando uma paleta de cores Neon (Sky-400) sobre fundos profundos (Slate-950), ideal para visualização prolongada sem fadiga ocular.
